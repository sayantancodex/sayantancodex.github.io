Projects

Infinity File Transfer Architecture (IFTA)

The need for secure and efficient file transfer across different operating systems has become increasingly important in today's digital age. This project presents the Infinity File Transfer Architecture, a secure cross-platform file transfer protocol that leverages encryption, decryption, and platform-specific tools like Netcat. The architecture ensures secure and reliable data transmission between Windows and Linux systems, with mathematical considerations for encryption and transmission efficiency.

sayOS Custom Linux Operating System

sayOS is a custom Linux operating system developed to support all terminal tasks while featuring a basic graphical user interface (GUI) with a resolution of 800x600. Built using open-source plugins like Openbox and Ly, sayOS includes an integrated Firefox browser despite limited graphics support. The kernel is based on Ubuntu 10, which is currently unsupported and buggy, making the OS not suitable for regular use. However, the creation of sayOS represents a significant personal achievement and a valuable learning experience in Linux and open-source software development.

Advanced Network Traffic Analysis and Anomaly Detection

This project investigates the analysis and anomaly detection of network traffic using packet capture (pcap) files. The primary objectives are to extract and analyze packet data, identify top talkers, analyze protocol distributions, detect known malicious IPs, and utilize machine learning algorithms to identify anomalies. The methods employed include statistical analysis, data visualization, and the application of the Isolation Forest algorithm for anomaly detection.

Automated Lung Tumor Segmentation using deep learning

Lung cancer remains a leading cause of cancer-related mortality worldwide, highlighting the necessity for early and accurate detection to improve patient outcomes. Manual segmentation of lung tumors from computed tomography (CT) images is labor-intensive and subjective, resulting in variability in results. This project develops a deep learning-based approach for lung tumor segmentation using the UNET model, known for its effectiveness in biomedical image segmentation. By automating the segmentation process, this study aims to enhance the precision and efficiency of diagnosis and treatment planning for lung cancer patients. The proposed method demonstrates significant improvements in segmentation accuracy and consistency, thereby reducing the workload of radiologists and improving patient care.

Pycypher

An encryption algorithm developed for securely encrypting and decrypting messages using a random key stored in a MySQL database.
The algorithm ensures robust security by leveraging Python libraries such as secrets, random, and pymysql.
The project's goal is to provide a user-friendly yet secure method for protecting textual data.

WebGuardian

WEBGUARDIAN is a cybersecurity tool designed to assess the security posture of websites by testing for various vulnerabilities and weaknesses. It provides a comprehensive analysis of web applications, focusing on critical aspects such as security headers, open ports, FTP access vulnerabilities, presence of admin panels, and susceptibility to common web attacks like XSS (Cross-Site Scripting) and SQL injection.
